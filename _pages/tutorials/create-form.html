Troubleshooting

Can't access Design Studio?

Make sure you're logged in and belong to an organization. If you haven't created an organization yet, follow this tutorial first.

Changes aren't saving?

Ensure you have a stable internet connection. Design Studio automatically saves your work, but network issues can interrupt the save process. Try refreshing the page to see if your changes persisted.

Can't find a specific field type?

Use the search bar in the field palette to quickly find field types. Common field types include text input, dropdown, date picker, file upload, and more.

Need more help?

Contact our support team at support@chorigen.com or visit the tutorials page for more guides.
